How to install git

Created2022-07-29|Updated2022-08-02|vscode运用
|Post View:

本次搭建git很简单

官方地址：https://git-scm.com/

1.Window版本点击Dwonload for Window，Mac点击下面的Mac Build就可以了

以window为例

2.里面有64位还有32位操作系统，我的电脑是64位，我选择了独立安装程序

3.双击下载好的git,进入安装

4.不需要点击only show new options,点击next

5.介绍：on the desktop是添加图标到桌面

  Window explorer integration下的两个功能勾选后是添加到鼠标右键菜单
  Git lfs是指大文件支持
  下面两个associate是指关联文件
  Check。。。是指检查git是否更新
 下面一个是将git bash的配置文件添加到windows终端 如图：

6.按照上图勾选，点击next

7.默认vim编辑器即可，点击next

8.默认选项即可，点击next

9.使用系统默认即可，点击next

10.使用捆绑的openssh,这里ssh.exe是git自带的

后面基本都是按照默认按next





11.注意这里的两项不需要勾选，直接点击install

12.安装成功标志

13.在更多中会显示

14.使用，可以在指定文件夹中执行相应命令
